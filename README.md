# Furyfish Studio

Website landing page cho Furyfish Studio - Studio phát triển game với phong cách nghệ thuật vẽ tay.

## 🌐 GitHub Pages

Website được host tại GitHub Pages.

## 📁 Cấu Trúc File

```
furyfish-studio/
├── index.html      # Trang HTML chính
├── styles.css      # CSS styles
├── main.js         # JavaScript (đa ngôn ngữ, load games)
├── games.json      # Danh sách game (dễ mở rộng)
├── assets/         # Thư mục chứa hình ảnh game
└── README.md       # Tài liệu
```

## 🎮 Thêm Game Mới

Mở file `games.json` và thêm game mới vào mảng:

```json
[
    {
        "name": "Tên Game",
        "url": "https://apps.apple.com/...",
        "image": "assets/your-game-image.png"
    }
]
```

## 🌍 Đa Ngôn Ngữ

Website hỗ trợ 3 ngôn ngữ:
- **English (EN)** - Mặc định
- **Tiếng Việt (VI)**
- **中文 (ZH)**

Để thêm/sửa bản dịch, chỉnh sửa object `translations` trong file `main.js`.

## ✨ Tính Năng

- **Phong cách nghệ thuật**: Thiết kế với mây trôi, đồi cỏ xanh
- **Responsive**: Hiển thị tốt trên mọi thiết bị
- **Dễ mở rộng**: Thêm game qua file JSON
- **Multi-language**: Hỗ trợ EN, VI, ZH
- **Animation**: Các hiệu ứng mượt mà, chuyên nghiệp
